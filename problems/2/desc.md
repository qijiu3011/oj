# 幸运数

**时间限制**: 1s &nbsp;&nbsp;|&nbsp;&nbsp; **内存限制**: 256MB &nbsp;&nbsp;|&nbsp;&nbsp;

## 题目描述

如果一个整数的十进制表示中只包含数字 **4** 和 **7**，则称这个数为**幸运数**。

例如：`4`、`7`、`47`、`477` 都是幸运数。

给定一个整数 n，判断 n 是否能被某个幸运数整除。

## 输入格式

一个整数 n（1 ≤ n ≤ 1000）。

## 输出格式

如果 n 能被某个幸运数整除，输出 `YES`，否则输出 `NO`。

## 样例

### 样例 1

输入：

```
47
```

输出：

```
YES
```

### 样例 2

输入：

```
17
```

输出：

```
NO
```

## 提示

1000 以内的幸运数有：

```
4, 7, 44, 47, 74, 77, 444, 447, 474, 477, 744, 747, 774, 777
```
